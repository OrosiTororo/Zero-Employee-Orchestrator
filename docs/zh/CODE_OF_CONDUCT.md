# 贡献者公约行为准则

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让每个人都能无骚扰地参与社区活动，无论其年龄、体型、残障状况、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、外貌、种族、宗教或性取向。

## 我们的标准

**积极行为：**
- 对他人表现出同理心和善意
- 尊重不同的观点、看法和经验
- 给予并优雅地接受建设性反馈
- 承认错误并从中学习
- 关注对整个社区最有利的事情

**不可接受的行为：**
- 使用色情语言或图像，或其他不当行为
- 恶意挑衅、侮辱性评论和人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息
- 其他在专业场合中被合理认为不当的行为

## 执行

社区领导者有责任明确和执行可接受行为的标准。对于被认为不当、威胁性、冒犯性或有害的行为，将采取适当和公正的纠正措施。

## 适用范围

本行为准则适用于所有社区空间，以及个人正式代表社区的场合。

## 举报

如果您目睹或遭受骚扰，请通过 [Issues](https://github.com/OrosiTororo/Zero-Employee-Orchestrator/issues) 举报。所有举报将得到及时公正的处理。

## 归属

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/) 2.1 版。
