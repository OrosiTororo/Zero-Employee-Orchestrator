## 2. 免费使用范围

通过以下方式可以实现完全免费运营。

### 方式 1：Ollama 本地模式

- **费用**：仅需电费
- **所需设备**：8GB+ 内存的 PC（推荐 16GB）
- **推荐模型**：`qwen3:8b`（8GB 内存）、`qwen3:32b`（32GB 内存）
- **限制**：模型性能可能不及云端 API

### 方式 2：Gemini 免费 API 密钥

- **费用**：免费（有速率限制）
- **获取方式**：[Google AI Studio](https://aistudio.google.com/)
- **限制**：每分钟 15 个请求，每天 1,500 个请求

### 方式 3：订阅模式（g4f）

- **费用**：使用现有的 ChatGPT Plus / Gemini Advanced 等订阅
- **限制**：稳定性可能较低。存在使用条款方面的风险

### 免费包含的功能

- 完整的 9 层架构
- Design Interview、Spec / Plan / Tasks
- Self-Healing DAG
- Judge Layer
- 审批流程
- 审计日志
- Experience Memory
- 全部 23 个 UI 界面
- Skill / Plugin / Extension 管理
- SQLite 数据库（开发和个人使用）

## 3. 硬件与环境限制

### 使用本地 LLM 时

| 模型大小 | 所需内存 | 推荐 GPU | 响应速度 |
|---|---|---|---|
| 1B～3B | 4 GB | 不需要 | 快速 |
| 7B～8B | 8 GB | 不需要（有则更快） | 中等 |
| 13B～14B | 16 GB | 推荐 VRAM 8GB 以上 | 较慢 |
| 30B～34B | 32 GB | 推荐 VRAM 16GB 以上 | 慢 |
| 70B+ | 64 GB+ | 需要 VRAM 24GB 以上 | 慢 |

### 数据库限制

| 数据库 | 并发连接数 | 数据大小上限 | 推荐用途 |
|---|---|---|---|
| SQLite | 1（写入） | 实际上无限制 | 个人使用・开发 |
| PostgreSQL | 数百 | 实际上无限制 | 团队・生产环境 |

### 网络

- 使用云端 LLM 时：需要互联网连接
- Ollama 本地模式：可完全离线运行
- WebSocket：实时通知需要浏览器连接

## 5. 大规模项目应用案例

**规模**：用 AI 团队替代 10 人市场营销团队的工作

**任务示例**：
- 自动创建每周 SNS 发布日历
- 定期生成竞争分析报告
- 撰写新闻稿草稿和多语言翻译

**预估费用**：每月 $50～$200（Sonnet + Flash 组合）

**所需环境**：

## 6. 成本优化建议

### 灵活使用执行模式

| 模式 | 用途 | 费用参考 |
|---|---|---|
| `quality` | 重要决策、对外发送前 | 高 |
| `speed` | 内部处理、草稿 | 中 |
| `cost` | 批量处理、常规任务 | 低 |
| `free` | 个人使用、测试 | 免费 |

### 推荐策略

1. **日常工作使用 `cost` 模式**：Haiku / Flash / Mini 即可提供足够的品质
2. **仅重要成果使用 `quality` 模式**：使用 Opus / GPT-5.4 进行最终确认
3. **Judge Layer 使用 `speed` 模式**：验证以速度为优先
4. **设置预算策略**：通过每日/每月上限防止意外费用
5. **本地模型与云端并用**：草稿在本地完成，润色在云端进行

### 预算管理功能

- 可设置每日/每周/每月预算上限
- 使用率达 80% 时发出警告，100% 时自动停止
- 通过成本台账进行交易级别的追踪
- 在仪表板上实时显示成本
