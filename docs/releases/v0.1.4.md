# v0.1.4 Release Notes

> Release date: 2026-04-07

## Cowork-Style Transition & First-Time User Experience

This release completes the transition from VSCode-style to Cowork-style terminology across the entire codebase, and addresses the top-priority UX issues identified in the v0.1.3 evaluation (7.7/10).

### New Features

**Operator Profile Page**
- New dedicated UI page at `/operator-profile` with two tabs: About Me and Global Instructions
- About Me: set your role, team, industry, responsibilities, priorities, and work style
- Global Instructions: persistent instructions injected into every AI conversation
- Connected to existing API endpoints (`PUT/GET /operator-profile/profile` and `/instructions`)
- Added to nav bar as a bottom item for quick access

**Welcome Tour**
- 3-step onboarding overlay for first-time users
- Step 1: "Describe tasks in natural language" (Dashboard)
- Step 2: "Monitor your AI agents" (Monitor)
- Step 3: "Control AI autonomy" (Autonomy Dial)
- Shows once, dismissible at any step, stored in localStorage
- Inspired by Claude Code's step-by-step quickstart pattern

**"What's New" Banner**
- Version-aware banner shown on Dashboard after updates
- Lists key highlights for the current version
- Dismissible, stored per-version in localStorage
- Inspired by Claude Code's release notes pattern
